# Distorting Embedding Space (DES)
Please refer [https://github.com/amoeba04/des](https://github.com/amoeba04/des), the official repository of **Mitigating Sexual Content Generation via Embedding Distortion in Text-conditioned Diffusion Models (NeurIPS 2025)**. (Distorting Embedding Space for Safety: A Defense Mechanism for Adversarially Robust Diffusion Models (ArXiv)) was a preprint of the NeurIPS paper.
